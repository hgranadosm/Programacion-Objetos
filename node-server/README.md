Para usar el servidor:

1. Ir a la carpeta node-server:
   cd node-server

2. Instalar dependencias:
   npm install

3. Iniciar el servidor:
   npm start

El servidor escuchará en http://localhost:3000

Los datos del formulario se enviarán a http://localhost:3000/api/prestamo y se imprimirán en la consola del servidor.
